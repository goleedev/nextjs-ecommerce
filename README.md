## 사용 방법

```
# Repository 복제하기
$ git clone https://github.com/goleedev/plask-assignment

# Repository로 이동하기
$ cd plask-assignment

# Dependencies 설치하기
$ npm install

# App 실행하기
$ npm run dev
```
